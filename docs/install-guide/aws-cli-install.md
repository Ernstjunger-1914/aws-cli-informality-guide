CLI 방식으로 AWS에 엑세스할 수 있는 계정이 없다면 [이 문서](https://github.com/Ernstjunger-1914/aws-cli-informality-guide/blob/docs/docs/install-guide/create-iam-account.md)를 참고바란다.
